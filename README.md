# 東美游泳隊官方網站

東美游泳隊官方網站以「每一滴汗水，都是金牌的序幕」為核心精神，呈現隊伍理念、教練團隊、升學路線、榮譽紀錄與招生資訊。網站同時提供獨立成績查詢頁，方便選手與家長快速查找相關公開成績。

## 網站連結

- 官方網站：`https://ray970225.github.io/dongmei-swim/`
- 成績查詢：`https://ray970225.github.io/dongmei-swim/results.html`

## 網站內容

### 首頁形象

首頁以東美泳隊品牌視覺作為第一印象，搭配深色水域背景、動態轉場與隊伍精神標語，建立具有辨識度的網站開場。

### 關於我們

介紹東美泳隊的訓練理念、團隊特色與在地發展方向，讓家長與選手能快速了解隊伍的核心價值。

### 升學路線

整理選手從國小、國中到高中階段的銜接路線，呈現東美泳隊在訓練與升學規劃上的完整培育方向。

### 最新動態

展示隊伍消息、活動公告與近期更新，讓使用者掌握東美泳隊的最新資訊。

### 教練團隊

呈現教練照片、職稱、學歷、經歷與專業證照，讓使用者認識每位教練的背景與專長。

### 榮譽殿堂

收錄選手比賽成果與獎牌紀錄，並提供分類與分頁瀏覽，讓榮譽資訊更清楚、易讀。

### 成績查詢

獨立成績查詢頁可輸入選手姓名，查詢東美泳隊相關公開成績。頁面設計以簡潔、快速、手機友善為主。

### 招生報名

提供招生資訊與報名表單連結，方便有興趣加入東美泳隊的家庭進一步聯繫。

## 設計方向

- 深色水域視覺，營造專業、沉穩與競技感。
- 使用亮青色與金色作為重點色，呼應游泳與獎牌意象。
- 頁面保留足夠留白，讓文字與卡片資訊更容易閱讀。
- 兼顧桌面端與手機端操作體驗。
- 以實用內容為主，不做過度裝飾，讓家長、選手與教練都能快速找到需要的資訊。

## 開發與維護

本網站由東美游泳隊維護，內容會依隊伍資訊、教練資料、比賽成績與招生需求持續更新。
